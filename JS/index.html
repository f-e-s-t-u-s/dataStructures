Linear Search (Sequential)

list of elements is searched for one by one, from the first to the last

The position of the element being searched for determines how fast the element is found

if the search is not successful, n comparisons are required. n is the number of elements in list

The time complexity is (O)n

Algorithm for Linear Search

       
       
        linearSearch(list, x) { 
            index = 0;
            found = true;
            for (index = 1; index < list.length; index++) {
                if (x == list[index]) {
                    return index;
                }
            }
            if (!found) {
                return "not found";
            }
        }
